TextConvertor

It converts text to binary numbers. Features:
- Option to select convert method (text to binary, text to ASCII)
- Option to save console output to a text file

Note: Placing a space in text input will cause errors so instead of space place "_"

Bugs:
-Placing space will cause errors in the program

Bugs fixed:
-Closing console before 5 seconds will cause the program to not save the text file if selected yet.
-File path not showing in console

If your antivirus is blocking this program convert the *.cpp file to exe and then run it
